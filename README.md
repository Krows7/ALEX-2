# ALEX-2 Minecraft Agent
Исследовательский проект по разработке пайплайна для агента [STEVE-1](https://arxiv.org/abs/2306.00937)

Авторы: Шага Константин, Спиряев Семен, Креславский Максим

Куратор: Кофанова Мария

# Цель проекта

Исследовать возможности ИИ для улучшения работы агента STEVE-1 и реализовать сервис для игры в Minecraft при помощи агента

# План работы

* Знакомство с куратором, постановка и уточнение задачи, план на год.
* Тестирование возможностей GPT-2 и анализ альтернатив.
* Анализ научных публикаций, связанных с данной темой. Систематизация данных и проведение EDA. Анализ обучаемых данных.
* Изучение подходов, применённых другими научными деятелями в схожих задачах. 
* Реализация интеграции выбранной модели в рабочее окружение. 
* Создание прототипа сервиса (веб-приложения или иного интерфейса) для удобного взаимодействия с моделью и настройки параметров обработки запросов.
* Проведение серии экспериментов с различными гипотезами, архитектурами и метриками для оценки эффективности модели. Сравнение результатов с первоначальной моделью (STEVE-1) и другими существующими решениями.
* Улучшение функционала и интерфейса сервиса по результатам тестирования. 
* Анализ итогов и подготовка отчёта
